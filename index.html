Stats & Chats

A project created in C++ that will parse over any text fed to it and generate a hashtable of the individual word frequencies within the text. From there, these frequencies are used to train a Markov process which allows the program to synthesize a new text that maintains the input's word frequencies and uses certain phrases found within the input text.
